# Projeto Final
Repositório para entrega do Projeto Final - Trabalho Final da disciplina Introdução à Análise Exploratória de Dados.
Projeto sobre a análise dos dados relativos ao covid-19 na cidade de Bauru.

Possui uma pasta com os dados no formato .csv e excel chamada "dados"

Possui uma pasta com os scripts em R chamada "codigos"

Possui uma pasta com todos os gráficos gerados chamado "graficos"

Possui um RMarkdown com a análise dos dados e sua versão em pdf na pasta "documentos"
